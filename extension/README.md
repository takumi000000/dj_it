# 🎧 DJ IT – YouTube DJ Mode (Chrome 拡張)

SpotifyのDJモード風に、**YouTube で似た曲調の曲を自動選曲し、2つのデッキでクロスフェード**しながら途切れなく繋ぐ Chrome 拡張機能 (Manifest V3) です。

## できること

- **全自動DJ**: 起動すると自動でミックス開始。似た曲調の選曲、クロスフェード、選曲の延長まで勝手に進行します。人の操作は「音を出す最初の1クリック」だけ（ブラウザ仕様上どうしても必要）。空のデッキに URL を貼るだけでもそのまま自動DJが始まります。
- **DJブース風ミキサーUI**: 2チャンネルのミキサー・コンソール。各デッキの**縦フェーダー（CH A / CH B）**と**MASTER**で音量、中央下の**クロスフェーダー**で A↔B のブレンドを手動操作できます。VUメーター風の表示付き（※装飾表示）。
- **集中モード / ブース窓**: 「🎛 集中」で映像を隠しミキサーだけの省スペース表示に。「🪟 ブース窓」で**小さな別ウィンドウに切り離し**（再生は途切れません）、作業画面の脇に置けます。
- **似た曲調の自動選曲**: 種になる曲から YouTube の Mix / ラジオ（`RD` プレイリスト）を生成し、YouTube 自身のジャンル・アーティスト類似レコメンドを選曲エンジンとして利用します。
- **デュアルデッキ・クロスフェード**: 2つの YouTube プレイヤー（デッキ A / B）をボリューム・クロスフェードで繋ぎ、DJ らしい滑らかな切り替えを実現します（フェード秒数は 2〜16 秒で調整可能）。
- **リクエスト割り込み**: YouTube URL を貼ると、自動ミックスに割り込んで次に再生できます。
- **YouTube ページ内ボタン**: 動画ページに「🎧 DJ IT で開始」ボタンを追加。今見ている曲を種に即スタート。
- **ポップアップ操作**: 拡張アイコンから「再生中の動画でDJ開始」「URL指定」「空のデッキ」を起動。

## 正直な技術的制約

YouTube の埋め込みプレイヤーは **生の音声波形 (PCM) にアクセスできない**ため、Spotify の DJ のような **BPM / キー解析による真のビートマッチはできません**。本拡張は現実的な代替として、

1. **似た曲調の選曲** = YouTube Mix (ラジオ) のレコメンド
2. **滑らかな繋ぎ** = ボリュームのクロスフェード

の組み合わせで「DJ モードらしい」体験を再現しています。

また Manifest V3 ではリモートスクリプト（`youtube.com/iframe_api`）を拡張ページで読み込めないため、YouTube 埋め込みプレイヤーの **postMessage プロトコルを自前実装**しています（`deck.js`）。

## インストール（開発者モード）

1. Chrome で `chrome://extensions` を開く
2. 右上の「デベロッパーモード」を ON
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. この `extension/` フォルダを選択

## 使い方

- **A) YouTube から**: 好きな曲の動画ページを開き、「🎧 DJ IT で開始」ボタン（または拡張アイコン →「再生中の動画でDJ開始」）をクリック。
- **B) ポップアップから**: 拡張アイコンをクリックし、URL を貼って「このURLでDJ開始」。

ブース画面が開くと、**DJが自動でミックスを開始します**（選曲・クロスフェード・選曲延長まで全自動）。ブラウザの自動再生ポリシー上、**音を出すための最初の1クリックだけ**が必要です（画面のどこをクリックしても、または上部の「🔊 クリックで音を出す」でOK）。それ以降は一切操作不要で流れ続けます。

- **CH A / CH B（縦フェーダー）**: 各デッキの音量
- **MASTER**: 全体音量
- **クロスフェーダー**: A↔B を手動でブレンド（自動ミックス中は自動で動きます）
- **⏭**: 今すぐクロスフェードで次の曲へ　／　**🔀**: 今の曲を種に Mix を再生成
- **XFADE / AUTO MIX**: クロスフェード秒数と自動ミックスの ON/OFF
- **🎛 集中 / 🪟 ブース窓**: 作業に集中したいときのコンパクト表示・切り離し

## ファイル構成

| ファイル | 役割 |
|---|---|
| `manifest.json` | MV3 マニフェスト |
| `deck.html` / `deck.css` / `deck.js` | DJ デッキ本体（デュアルデッキ・クロスフェードエンジン、YouTube postMessage 実装） |
| `content.js` / `content.css` | YouTube 動画ページに起動ボタンを注入 |
| `popup.html` / `popup.js` | ツールバーのポップアップ UI |
| `background.js` | デッキタブを開く Service Worker |
| `icons/` | アイコン |

## テスト

- `deck.js` の純粋ロジック（URL パース・キュー制御）のユニットテスト
- スタブ化した YouTube 埋め込みでの E2E（prepare → Mix 取得 → 再生 → クロスフェードでデッキ切替 → リクエスト割り込み）

いずれもローカルで検証済みです。実際の YouTube に対する動作は、上記手順で拡張を読み込んで確認してください。
